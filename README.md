# Student Performance Prediction System

## 📌 Project Overview
This project predicts whether a student will pass or fail based on academic performance factors using Machine Learning algorithms.

---

## 📌 Features Used
- Weekly Self Study Hours
- Attendance Percentage
- Class Participation
- Total Score

---

## 📌 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

---

## 📌 Machine Learning Algorithm
- Logistic Regression

---

## 📌 Features
- Data Preprocessing
- Student Performance Prediction
- Accuracy Evaluation
- Confusion Matrix
- Visualization

---

## 📌 Output
The system predicts student performance and helps identify weak students for academic improvement.

Model Accuracy: High Accuracy Classification Model
